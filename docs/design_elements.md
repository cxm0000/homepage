## Typography

Use a modern, professional sans-serif font with excellent readability across web, iOS, and Android.

Recommended primary font:

- **Inter** for web and app UI

Fallback stack:

```css
font-family: Inter, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif;
```

## Discoverability (SEO and AI agents)

Typography and layout choices should support **readability for humans and parsers**: clear hierarchy (one H1, ordered section headings), comfortable line-height, and real text for key claims—not only graphics. Full principles (meta tags, semantic HTML, anchors, optional `llms.txt`) live in **`optionwise_homepage_design_guidance.md` §22**.

## Typography style

* Use strong but calm headings.
* Avoid overly playful or rounded fonts.
* Keep the product feeling professional, trustworthy, and B2B-focused.
* Use generous line-height for homepage copy.
* Avoid too many font weights.

Suggested font weights

* Hero heading: 700 or 800
* Section headings: 700
* Body text: 400 or 500
* Labels / eyebrow text: 600 or 700 with increased letter spacing
* Buttons: 600

Suggested sizes

Desktop:

* Hero heading: 56–72px
* Hero body: 18–20px
* Section heading: 36–48px
* Card title: 18–22px
* Body text: 15–17px
* Navigation: 14–15px

Mobile:

* Hero heading: 36–44px
* Hero body: 16–18px
* Section heading: 28–34px
* Body text: 15–16px
